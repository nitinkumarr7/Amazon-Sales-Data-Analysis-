# Amazon-Sales-Data-Analysis-
Amazon Sales Data Analysis Dashboard

Project Overview

This project focuses on analyzing Amazon sales data using Python to uncover valuable business insights related to sales performance, profitability, customer purchasing behavior, and product trends.

The dataset was sourced from Kaggle and contains sales-related information such as orders, products, categories, sales, quantity, and profit. Before analysis, the dataset was cleaned and prepared to ensure accurate and reliable results.

Data Cleaning & Preparation

- Checked the dataset for missing values and data quality issues
- Cleaned and organized the data for analysis
- Verified data types and column consistency
- Handled data inconsistencies and prepared the dataset for analysis
- Ensured data accuracy before generating insights

Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

Project Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze sales, quantity, and profit performance
- Identify top-performing products and categories
- Evaluate monthly and yearly sales trends
- Analyze regional and category-wise profitability
- Generate actionable business insights

Key Insights

- Identified top-performing products and categories
- Analyzed sales and profit performance across different regions
- Evaluated trends and patterns in customer purchasing behavior
- Discovered opportunities for improving profitability
- Generated data-driven insights to support business decision-making

Conclusion

This project demonstrates a complete data analysis workflow, including data cleaning, exploratory data analysis, data visualization, and business insight generation using Python. The analysis helps understand sales performance and supports informed business decisions.
